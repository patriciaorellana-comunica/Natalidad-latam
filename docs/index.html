Análisis Demográfico de Natalidad en LATAM (2001–2023)

Autor: Tu Nombre — Fecha: 2026

Este proyecto analiza la evolución de las tasas de natalidad en América Latina entre 2001 y 2023, con el objetivo de demostrar un flujo de trabajo reproducible de limpieza de datos, procesamiento estadístico y ranking de indicadores demográficos para la comunicación de datos y el análisis de políticas públicas.
